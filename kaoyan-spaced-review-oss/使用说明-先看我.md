# 考研间隔复习 Skill 使用说明

这是一个给 2027 考研数学一和信号与系统用的多 Agent 复习系统。

## 书放哪里

把教材 PDF 放到下面这些文件夹：

- 高数张宇教材：`assets/textbooks/higher_math/`
- 线代杨威教材：`assets/textbooks/linear_algebra/`
- 概率论方浩教材：`assets/textbooks/probability/`
- 信号与系统教材：`assets/textbooks/signals_systems/`

把后续强化、真题、讲义放到下面这些文件夹：

- 数学基础资料：`assets/materials/math_foundation/`
- 数学强化资料：`assets/materials/math_intensive/`
- 数学真题/模拟：`assets/materials/math_past_papers/`
- 信号基础资料：`assets/materials/signals_foundation/`
- 信号强化资料：`assets/materials/signals_intensive/`
- 信号真题/模拟：`assets/materials/signals_past_papers/`

## 推荐文件名

文件名可以用中文，例如：

- `张宇高数2027.pdf`
- `杨威线代2027.pdf`
- `方浩概率论2027.pdf`
- `信号与系统教材.pdf`
- `数学强化讲义.pdf`
- `专业课真题.pdf`

## 第一次使用

在 Codex 里说：

```text
用 $kaoyan-spaced-review，先检查我放进去的 PDF 质量，判断哪些需要 OCR 或人工复核。
```

检查完 PDF 后，再说：

```text
用 $kaoyan-spaced-review，根据教材给我制定 2027 考研数学一和信号与系统阶段规划。
```

开始抽卡时说：

```text
用 $kaoyan-spaced-review，从高数第一章开始，按多 agent 流程生成知识点、公式推导、填空题和书上例题变式复习卡。
```

## 每天复习

每天可以说：

```text
用 $kaoyan-spaced-review，生成今天的复习任务，先问我题，不要直接给答案。
```

答完后，用 `again`、`hard`、`good`、`easy` 记录结果：

- `again`：完全不会或忘了
- `hard`：想起来了但很费劲
- `good`：基本掌握
- `easy`：非常熟

系统会根据结果安排下次复习。

## 每天自动推送

告诉 Codex 你想每天几点收到复习，例如：

```text
每天早上 8 点，用 $kaoyan-spaced-review 给我推送今日复习。
```

Codex 会创建一个自动化任务，每天唤醒这个线程并生成复习内容。
