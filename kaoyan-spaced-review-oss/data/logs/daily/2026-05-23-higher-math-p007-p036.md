## 一、逐页知识点清单

### PDF p.7 / 印刷 p.1

- 第1讲主题：函数极限与连续；考题核心是计算函数极限，题型覆盖选择、填空、解答。
- 本讲目标覆盖：函数概念、表示法、应用函数关系；有界性、单调性、周期性、奇偶性；复合函数、分段函数、反函数、隐函数；基本初等函数性质与图形；极限概念、性质、左右极限；极限四则运算、洛必达、泰勒、两个重要极限；无穷小、无穷大、等价无穷小；连续与间断。
- 重点提示：洛必达法则、泰勒公式。

### PDF p.8 / 印刷 p.2

- 第1讲知识结构：函数的概念与特性、函数图像、函数极限的概念与性质、极限计算、函数连续与间断。
- 极限计算分支：四则运算、洛必达、泰勒、无穷小运算、两个重要极限、夹逼准则、七类未定式。
- 学习要求：先建立框架，再逐页填充细节。

### PDF p.9 / 印刷 p.3

- 函数定义：给定数集 D，对每个 x in D，按同一法则有唯一确定的 y 与之对应，记作 y=f(x)。
- 术语：x 是自变量，y 是因变量，D 是定义域，{f(x) | x in D} 是值域。
- 单值函数与多值函数：一个 x 对应一个 y 是单值函数；一个 x 对应多个 y 是多值函数。
- 图像判别：竖直线法。
- 例1.1：由 f(x+1/x) 的表达式反推出 f(x)，关键是把 x+1/x 作为整体。

### PDF p.10 / 印刷 p.4

- 例1.1 方法总结：把复合输入整体化，寻找对应法则。
- 常用恒等式：x^2 + 1/x^2 = (x + 1/x)^2 - 2。
- 例1.2：由含 f(x) 与 f(1/x) 的关系式求 f(x)。
- 例1.2 方法：把 x 替换为 1/x，建立第二个方程，联立消去 f(1/x)。
- 方法总结：用所给关系式建立方程组，求所需函数值或表达式。

### PDF p.11 / 印刷 p.5

- 反函数定义：若 y=f(x) 的值域 R 中每个 y 都对应唯一 x，则可定义反函数 x=f^{-1}(y)。
- 反函数的定义域是原函数值域，值域是原函数定义域。
- 严格单调函数必有反函数；有反函数不一定单调。
- 图像关系：写成 y=f^{-1}(x) 后，与 y=f(x) 关于 y=x 对称。
- 判别反函数：水平线法。
- 重要关系：f(f^{-1}(x))=x，f^{-1}(f(x))=x。
- 例1.3：求 y=ln(x+sqrt(x^2+1)) 的反函数及定义域。

### PDF p.12 / 印刷 p.6

- 例1.3 方法：利用对数性质，有理化，构造 e^y 与 e^{-y} 两式相减，反解 x。
- 重要函数：ln(x+sqrt(x^2+1)) 是反双曲正弦函数；(e^x-e^{-x})/2 是双曲正弦函数；(e^x+e^{-x})/2 是双曲余弦函数。

### PDF p.13 / 印刷 p.7

- 关于 ln(x+sqrt(x^2+1)) 的三个结论：x->0 时与 x 等价；导数为 1/sqrt(x^2+1)；它是奇函数，相关对称性可用于积分。
- 复合函数定义：若 y=f(u)，u=g(x)，且 g(D) 在 f 的定义域内，则 y=f(g(x)) 是复合函数。
- 例1.4：已知 f(x)=x^2 与 f(phi(x)) 的表达式，求 phi(x) 及定义域、值域。
- 方法：把 phi(x) 看成整体，先反推，再由根号内非负确定定义域和值域。

### PDF p.14 / 印刷 p.8

- 例1.5：分段函数复合，求 g[f(x)]。
- 方法：先根据内层函数 f(x) 的取值，决定外层 g 的分段公式；可画图辅助判断。
- 隐函数定义：方程 F(x,y)=0 在某区间内对每个 x 有唯一 y 值，则确定隐函数 y=y(x)。
- 隐函数可显化或难以显化；特殊点可用观察法。

### PDF p.15 / 印刷 p.9

- 有界性：在区间 I 上存在 M>0，使任意 x in I 都有 |f(x)| <= M，则 f 在 I 上有界。
- 必须指明讨论区间；同一函数在不同区间可有界或无界。
- 几何理解：图像能被 y=M 与 y=-M 夹住。
- 若区间内或端点附近函数趋于无穷大，则无界。
- 例1.6：证明 f(x)=x/(1+x^2) 在全实数上有界。

### PDF p.16 / 印刷 p.10

- 例1.6 方法：用基本不等式找统一上界；由 1+x^2 >= 2|x| 得 |x|/(1+x^2) <= 1/2。
- 单调性：x1<x2 时 f(x1)<f(x2) 为单调增加；f(x1)>f(x2) 为单调减少。
- 判别式：严格增对应 (x1-x2)[f(x1)-f(x2)]>0；严格减对应 <0；不减、不增对应 >=0、<=0。
- 例1.7：已知 f 严格单调增加，判断 |f(x)|、f(|x|)、f(-x)、-f(-x) 哪个一定单调增加。

### PDF p.17 / 印刷 p.11

- 例1.7 图像变换：|f(x)| 是 x 轴翻折；f(|x|) 是右半图向左镜像；f(-x) 关于 y 轴；-f(-x) 关于原点。
- 结论：-f(-x) 一定单调增加。
- 图像公式：f(x) 与 f(-x) 关于 y 轴；f(x) 与 -f(x) 关于 x 轴；f(x) 与 -f(-x) 关于原点。

### PDF p.18 / 印刷 p.12

- 奇偶性前提：定义域关于原点对称。
- f(-x)=f(x)：偶函数；f(-x)=-f(x)：奇函数。
- f(x)+f(-x) 必为偶函数；f(x)-f(-x) 必为奇函数。
- 任意函数可拆成偶部 1/2[f(x)+f(-x)] 与奇部 1/2[f(x)-f(-x)]。
- 复合函数的奇偶性要看内外层奇偶性。

### PDF p.19 / 印刷 p.13

- 常见奇偶复合：奇[偶]->偶；偶[奇]->偶；奇[奇]->奇；偶[偶]->偶；非奇非偶[偶]->偶。
- ln(x+sqrt(x^2+1)) 是奇函数；其导数为偶函数，二阶导数为奇函数。
- 若 f 为奇函数，则 integral_0^x f(t)dt 为偶函数。
- 例1.8：由 f(x+y)=f(x)+f(y) 证明 f 为奇函数；先令 x=y=0，再令 y=-x。
- 周期性定义：存在 T>0，使 f(x+T)=f(x)，则 f 为周期函数。

### PDF p.20 / 印刷 p.14

- 周期结论：f(ax+b) 的周期为 T/|a|；复合 f(g(x)) 可能周期；周期函数的导函数仍周期；积分函数要周期需满足一周期积分为 0。
- 例1.9：由 f(x)=f(x-pi)+sin x 证明 f 以 2pi 为周期。
- 函数图像开始：基本初等函数包括常数、幂、指数、对数、三角、反三角函数。
- 常数函数 y=A：图像为水平直线。

### PDF p.21 / 印刷 p.15

- 幂函数 y=x^mu；当 x>0 时对任意实数 mu 有定义。
- 常见幂函数：x、x^2、sqrt(x)、x^3、cuberoot(x)、1/x。
- 极值技巧：根式可转为研究内部；绝对值可转为平方；乘积可取 ln；1/u 与 u 的最值反向。
- 例1.10：求 x^6(1-x)^2(1-2x)^4 在 0<x<1/2 上的最大值。

### PDF p.22 / 印刷 p.16

- 例1.10 方法：对乘积幂函数取对数求导，排除不符合区间的根，并检查端点极限。
- 公式：(ln x)'=1/x；[ln(1-x)]'=1/(x-1)；[f(g(x))]'=f'(g(x))g'(x)。
- 指数函数 y=a^x，a>0,a!=1：定义域 R，值域 (0,+infty)；a>1 增，0<a<1 减；常用 e^x。

### PDF p.23 / 印刷 p.17

- e^x 极限：x->-infty 时趋 0；x->+infty 时趋 +infty。
- 特殊值：a^0=1，e^0=1。
- 指数运算规则：同底相乘、相除、乘方、积的乘方、商的乘方。
- 对数函数 y=log_a x：是 y=a^x 的反函数；定义域 (0,+infty)，值域 R；a>1 增，0<a<1 减；常用 ln x。
- 特殊值：log_a1=0，log_a a=1，ln1=0，lne=1。

### PDF p.24 / 印刷 p.18

- ln x 极限：x->0+ 时趋 -infty；x->+infty 时趋 +infty。
- 常用变形：x=e^{ln x}；u^v=e^{v ln u}；x^x=e^{x ln x}。
- 对数运算：积、商、幂、根式的对数转化。
- 常见化简：ln sqrt(x)=1/2 ln x；ln(1/x)=-ln x；ln(1+1/x)=ln(x+1)-ln x。
- 例1.11：由 e^x 级数展开求 2^x 展开。

### PDF p.25 / 印刷 p.19

- e^x 展开：e^x=sum x^n/n!。
- 2^x=e^{x ln2}，代入展开即可。
- sin x 与 cos x：定义域 R，值域 [-1,1]；sin 为奇，cos 为偶；最小正周期均为 2pi；均有界。
- 常用特殊角：0、pi/6、pi/4、pi/3、pi/2、pi、3pi/2、2pi 的 sin/cos 值。

### PDF p.26 / 印刷 p.20

- 恒等式：sin^2 alpha + cos^2 alpha=1。
- tan x=sin x/cos x；cot x=cos x/sin x=1/tan x。
- tan 定义域 x != kpi+pi/2；cot 定义域 x != kpi；值域均 R。
- tan、cot 均为奇函数，最小正周期均为 pi。
- 特殊值和靠近渐近线的极限需要熟记。

### PDF p.27 / 印刷 p.21

- sec x=1/cos x；csc x=1/sin x。
- sec 定义域 x != kpi+pi/2；csc 定义域 x != kpi。
- 值域均为 (-infty,-1] union [1,+infty)。
- sec 为偶函数，csc 为奇函数；最小正周期均 2pi。
- 恒等式：1+tan^2 alpha=sec^2 alpha；1+cot^2 alpha=csc^2 alpha。
- arcsin、arccos 开始：分别是 sin 在 [-pi/2,pi/2]、cos 在 [0,pi] 上的反函数。

### PDF p.28 / 印刷 p.22

- arcsin 主值区间 [-pi/2,pi/2]；arccos 主值区间 [0,pi]。
- 恒等式：sin(arcsin x)=x；cos(arccos x)=x；sin(arccos x)=sqrt(1-x^2)；cos(arcsin x)=sqrt(1-x^2)。
- arcsin(sin x)=x 只在主值区间；arccos(cos x)=x 只在主值区间。
- arcsin x + arccos x = pi/2。
- 特殊值：arcsin/arccos 在 0、1/2、sqrt2/2、sqrt3/2、1 处的值。
- 例1.12：对 y=sin x 在 [0,2pi] 上分单调区间求反函数。

### PDF p.29 / 印刷 p.23

- 例1.12 方法：分单调区间，分别回到 arcsin 主值区间。
- sin x 在 [0,2pi] 上不能直接写单一普通反函数，否则得到多值关系。
- arctan x：tan x 在 (-pi/2,pi/2) 上的反函数。
- arccot x：cot x 在 (0,pi) 上的反函数。

### PDF p.30 / 印刷 p.24

- arctan/arccot 定义域均为 R。
- arctan 值域 (-pi/2,pi/2)，单调增；arccot 值域 (0,pi)，单调减。
- arctan 为奇函数；arctan x + arccot x = pi/2。
- 两者均有界；无穷远处有对应极限。
- 初等函数：由基本初等函数经有限次四则运算和有限次复合、并能由一个式子表示。

### PDF p.31 / 印刷 p.25

- 初等函数定义域可以是区间、区间并集或孤立点集合。
- 幂指函数 u(x)^{v(x)} 可写成 e^{v(x)ln u(x)}，在 u(x)>0 时是初等函数。
- 分段函数：自变量不同范围内由不同表达式表示，但仍是一个函数。
- 三个重要分段函数：|x|、sgn x、[x]。
- sgn x 与绝对值关系：x=|x|sgn x。
- [x] 是不超过 x 的最大整数；定义域 R，值域 Z，在整数点跳跃。

### PDF p.32 / 印刷 p.26

- 取整函数性质：[x+n]=[x]+n；x-1<[x]<=x；x->0+ 时 [x]->0，x->0- 时 [x]->-1。
- 例1.13：y=x-[x] 是小数部分函数，以 1 为周期。
- 图像应用：|sin x|、|arcsin(sin x)| 等图像在积分题中常出现。

### PDF p.33 / 印刷 p.27

- 函数极限的概念与性质开始。
- 邻域：U(x0,delta)={x | |x-x0|<delta}。
- 去心邻域：0<|x-x0|<delta。
- 右邻域、左邻域分别只取右侧或左侧。
- 邻域用于描述点附近的局部行为。

### PDF p.34 / 印刷 p.28

- 函数极限定义：f 在 x0 某去心邻域有定义；若任意 epsilon>0 都存在 delta>0，使 0<|x-x0|<delta 时 |f(x)-A|<epsilon，则 lim f(x)=A。
- epsilon 控制函数值距离；delta 控制自变量距离。
- forall 表示任意；exists 表示存在。
- 图像理解：epsilon 带越窄，仍能找到足够小的 delta 邻域让函数图像落入带内。

### PDF p.35 / 印刷 p.29

- 极限定义共有 24 种形式：6 种自变量趋近方式乘 4 种函数趋近方式。
- 自变量趋近方式：x->x0，x->x0+，x->x0-，x->infty，x->+infty，x->-infty。
- 函数趋近方式：f(x)->A，f(x)->infty，f(x)->+infty，f(x)->-infty。
- 有限极限用 epsilon；无穷极限用 M；x 趋于无穷时用 X。
- 例1.14 开始：设未知极限 A，利用题设关系求 A。

### PDF p.36 / 印刷 p.30

- 例1.14 方法：设 A=lim f(x)/x^2；把题中等式除以 x^2，转成关于 A 的方程。
- 使用极限：(x-sin x)/x^3 -> 1/6，x^2/(1-cos x)->2。
- 超实数应用开始：
  - R 中不存在非零无穷小及其倒数无穷大量。
  - 非零无穷小 x*：任意 n 都有 |x*|<1/n 且 x* != 0。
  - 1/x* 是无穷大量。
  - 超实数系 R* 由实数、非零无穷小、无穷大量构成。
  - x0+x* 是有限超实数，x0+1/x* 是无穷大超实数。
